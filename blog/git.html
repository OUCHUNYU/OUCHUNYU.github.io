What are the benefits of version control?

To understand what Version Control benefits programmers and what Version Control is, is essential for a software developer to become a better team player in real world tasks. A simple way to understand Version Control is that Version Control keeps track of changes of a file or files during a file or files lifetime. It is simple and useful for everyday tasks. Either you work alone or with others, it is good to use Version Control, because there are several benefits it offers us. Benefit#1: Version Control keeps track of your files and can tell all the changes you made and why you made them and when you made them. In case something goes wrong you are able to comeback to an earlier point to solve the problems. Benefit#2: Version Control allows you to have more than one person working on the same file at the same time. Benefit#3: It generates a new set of files for eachone who works with you, no one will override somebody else's work. So use version control and have a clear work flow.

How does git help you keep track of changes?

There are quite few different version control systems. Here I’m only gonna introduce you Git. Note Git is not GitHub. Git is a version control system software enables all the benefits of Using Version Control. It gives you ability to track changes, and make changes. Git unifies collaborative changes into commits, and allows you to take commits from more than one person and merge them into the project. It is simple, fast and lightweight. Once all tasks done it will get out the way immidiately. Git provides a tree like architecture for a project which allows a project to have many branches, so we can collaborate with more people. Git is something truly brilliant!

Why use GitHub to store your code?

To understand Why use GitHub to store your code is to understand what GitHub does for us. GitHub is a website where developers can share their code and keep track of changes using Git remotely. Github makes open source truly open, that people from all around the world can see. It provides awesome user interface to see when the changes were made. Because you need an account to use GItHub so it can also tell who made the changes. GitHub takes Version Control to a remote level, you can store your code and share with people or your collaborators, and allows them to contribute, or just to get some awesome ideas from other Github users.

Overall, either Git or GitHub, are powered by Version Control, so start using it and happy coding!
